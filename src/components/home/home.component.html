Hello Xello Team..

This is Angular application with single page or view.

This Angular application is created using Angular CLI boiler plate.

This application is built with simple routing into other components and is styled with Scss BEM.

The two buttons are placed in the contact component, where all the given conditions for the tooltip has been completed.

This application can be run with the single npm command.

I've used high contrast colours in application, which is one good Accessibility feature i've used.
